Browser Targeting

You can target different versions of Internet Explorer from within your CSS. This could have been made to target any browser or version, but that would promote bad CSS habits. However, targeting IE is a common task for a web designer, and this makes the process much more transparent.

The boxes below will change color depending on which browser you are using. The best way to test this is to use Safari 4 and change the User Agent in the develop menu.
