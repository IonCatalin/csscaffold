h2. columns-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.0|1.1|

This class makes the element function as a column. It adds, floats, margins etc. See the columns property for more details.

*Example*

<pre><code><div class="columns-4"></div></code></pre>

h2. span-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.1|-|

Gives the element a width of x number of columns.

*Example*

<pre><code><div class="span-4"></div></code></pre>

h2. push-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.0|1.1|

Pushes a column to the right using margins.

*Example*

<pre><code><div class="columns-8 push-4"></div></code></pre>

h2. pull-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.0|1.1|

Pulls a column to the left using margins.

*Example*

<pre><code><div class="columns-8 pull-4"></div></code></pre>

h2. append-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.0|-|

Appends x number of columns using padding.

*Example*

<pre><code><div class="columns-8 pull-4 append-1"></div></code></pre>

h2. prepend-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.0|-|

Prepends x number of columns using padding.

*Example*

<pre><code><div class="columns-8 pull-4 prepend-1"></div></code></pre>

h2. baseline-x

|_.Plugin|_.Added|_.Updated|
|Layout|1.0|-|

Adds the height of x number of baselines. x can be anything up to the total column-count.

*Example*

<pre><code><div class="baseline-2"></div></code></pre>
