h2. small-type

|_.Param|_.Default value|_.Description|_.Values|
|n/a|-|-|-|

Makes the text use incremental leading and decreases its size. Every 4 lines of normal type, there are 5 lines of smaller type.

Code:
<pre><code>=small-type
{ 
	line-height: eval(floor((!baseline*4)/5))px; 
	font-size: 11px; 
}
</code></pre>

h2. all-caps

|_.Param|_.Default value|_.Description|_.Values|
|n/a|-|-|-|

Surround uppercase words and abbreviations with this class. Based on work by Jørgen Arnor Gårdsø Lom [http://twistedintellect.com/]

Code:
<pre><code>
	=all-caps
	{ 
		font-variant: small-caps; 
		letter-spacing: 1px; 
		text-transform: lowercase; 
		font-size:1.2em;
		line-height:1%;
		font-weight:bold;
		padding:0 2px;
	}
</code></pre>

|n/a|-|-|-|

h2. image-replaced

|_.Param|_.Default value|_.Description|_.Values|
|n/a|-|-|-|

Pushes the text out of view. It requires that the element also has a height, width and a background image. Remember that Scaffold comes with a built-in image replacement plugin.

You can image replace like so:

<pre><code>
.selector 
{
	image-replace: 'image-name'; 
}
</code></pre>

Code:
<pre><code>
   	=image-replaced
	{
		display:block;
		text-indent:-9999px;
		background-repeat: no-repeat;
		background-position: 0 0;
		overflow:hidden;
	}
</code></pre>
